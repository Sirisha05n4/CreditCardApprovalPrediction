| Model Development | As a developer, I want to train a Random Forest model so applications can be classified as Approved/Rejected | High |
| Backend API | As a developer, I want a Flask endpoint that accepts applicant data and returns a prediction | High |
| Frontend UI | As a user, I want a simple form to enter my details and see my result clearly | High |
| Result Display | As a user, I want to see a confidence score along with the decision so I understand how certain the prediction is | Medium |
| Deployment | As a team, we want the app running locally (and optionally publicly) so it can be demoed | Medium |
